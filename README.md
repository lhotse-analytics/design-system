## Prerequisites

You will need the following installed on your computer.

* [Git](https://git-scm.com/)
* [Node.js v18+](https://nodejs.org/) (with npm)
* Text Editor
* Terminal
* Browser

## Installation

- `git clone <repository-url>` this repository
- `cd <project-directory>`
- `npm install`
- `npm run docs:dev`
